# Bibliometric Analysis App

## Overview

This project is a Streamlit-based application designed to perform bibliometric analysis on academic publication data. The app enables users to explore research trends, keyword relationships, and publication patterns through interactive visualizations.

## Features

* Publication trend analysis over time
* Keyword frequency and co-occurrence analysis
* Interactive data visualization
* Simple interface for exploring bibliometric datasets

## Technology

* Python
* Streamlit
* Pandas
* Plotly / Matplotlib

## Usage

1. Upload or load bibliometric dataset (CSV format).
2. Select analysis options from the sidebar.
3. Explore publication trends, keyword networks, and other visual outputs.

## Deployment

This application is deployed using Streamlit Cloud.
